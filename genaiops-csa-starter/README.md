# Public GenAIOps Starter for Cloud Solution Architects

A public-safe learning package for new and aspiring Cloud Solution Architects who want to understand how generative AI moves from prototype to production.

This project is designed for publication on rbcloud.co.uk. It avoids customer-specific details, internal workshop language, private account context, and confidential Microsoft/customer information. It uses general Azure architecture patterns and public-facing explanations.

## Audience

- New CSA starters
- Community members learning Azure AI architecture
- Architects who need a practical GenAIOps operating model
- Technical sellers and consultants who need better customer discovery questions

## Positioning

Working title: **GenAIOps for Cloud Solution Architects: From Prompt to Production**

Tagline: **A practical public guide to building, evaluating, deploying, monitoring, governing, and optimising GenAI solutions on Azure.**

## Public-safety principle

Do not publish customer-specific architecture, customer names, customer workshop content, internal field guidance, confidential commercial details, or verbatim private deck content. The public version should teach reusable concepts and patterns only.
