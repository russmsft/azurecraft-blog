# Video Production Brief

## Working title

GenAIOps for Cloud Solution Architects: From Prompt to Production

## Purpose

Create a public, community-friendly explainer that helps new CSA starters understand how to talk about operationalising GenAI on Azure.

## Audience

- New Cloud Solution Architects.
- Community learners.
- Architects moving from cloud infrastructure into AI applications.
- Technical sellers and consultants who need a practical conversation model.

## Format

Recommended length: 8 to 10 minutes.

Style: calm, practical, architect-to-architect. Avoid hype. Use simple visuals, lifecycle diagrams, and a few realistic but generic examples.

## Structure

1. Hook: GenAI demos are easy; production AI is harder.
2. Define GenAIOps.
3. Walk through the six lifecycle stages.
4. Explain the Azure reference pattern.
5. Cover evaluation and monitoring.
6. Explain governance, AI gateway, and cost.
7. Close with a practical pilot checklist.

## Visual style

Match rbcloud.co.uk: terminal-inspired, dark background, clean cards, minimal animation, diagram-first explanations.

## Public-safety rule

Do not show customer screenshots, internal decks, private tenant details, customer names, or confidential account context. Use simple generated diagrams and public-safe examples only.

## AI voice disclosure

If using AI-generated narration based on your own voice, include a short disclosure in the video description and optionally at the end of the video:

"Narration uses an AI-generated version of my own voice, created with my consent. The script and opinions are my own."
